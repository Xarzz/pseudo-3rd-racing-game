# Assets Folder

Folder ini berisi asset untuk kendaraan pemain.

## Cara Menambahkan Asset Kendaraan

1. Simpan file gambar kendaraan dengan format berikut:
   - `car-1.png`
   - `car-2.png`
   - `car-3.png`
   - `car-4.png`
   - `car-5.png`
   - `car-6.png`

2. Ukuran yang disarankan: 150x300 pixel (portrait)
3. Format: PNG dengan background transparan
4. Posisi: Menghadap ke atas

## Struktur File

```
/public/assets/vehicles/
├── car-1.png
├── car-2.png
├── car-3.png
├── car-4.png
├── car-5.png
└── car-6.png
```
